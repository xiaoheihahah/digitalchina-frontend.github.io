# 首页配置

# 介绍

VuePress 的默认主题提供了一套主题模板，可以通过在 Markdown 文件顶部配置 YAML在新窗口打开 Frontmatter 指明该页面为主页以应用主题模板：

---
home: true
---

首页中可以配置的 Frontmatter API 可以参考 Frontmatter 首页在新窗口打开，除了 Frontmatter 外的 Markdown 内容会被渲染至主题模板之后。
